# 春夜喜雨 - 杜甫

## 项目说明
这是一个基于杜甫《春夜喜雨》创建的交互式网页，采用现代化的设计和动画效果。

## 特色功能
- 🌧️ 动态雨滴效果
- ☁️ 浮动云朵动画
- 🌸 花朵装饰元素
- ⭐ 闪烁星星效果
- 🎨 深色主题设计
- 📱 响应式布局
- ✨ 诗句交互效果

## 技术栈
- HTML5
- CSS3 (动画、渐变、滤镜)
- JavaScript (交互效果)
- 无需外部依赖

## 浏览器支持
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 使用方法
直接在浏览器中打开 `index.html` 文件即可查看效果。

## 诗歌原文
```
好雨知时节，当春乃发生。
随风潜入夜，润物细无声。
野径云俱黑，江船火独明。
晓看红湿处，花重锦官城。
```
